# vue-music

> Vue.js ——音乐 App


### 安装与运行

```
git clone https://github.com/dingyanhe/myVueMusic.git

cd myVueMusic

npm install // 国外镜像会很慢

npm start //服务端运行 访问 http://localhost:8899
```
